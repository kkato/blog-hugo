### nvim-tree

| キー | 動作 |
|------|------|
| `<Space>e` | ツリー開閉 |
| `<Space>o` | 現在のファイルを表示 |
| `g?` | ヘルプ |
| `<CR>` / `o` | 開く |
| `<C-v>` | 垂直分割で開く |
| `<C-x>` | 水平分割で開く |
| `<C-t>` | タブで開く |
| `a` | 作成 |
| `d` | 削除 |
| `r` | リネーム |
| `x` / `c` / `p` | 切り取り / コピー / 貼り付け |

### ペイン操作

| キー | 動作 |
|------|------|
| `<C-w>h/j/k/l` | 左/下/上/右へ移動 |
| `<C-w>c` | 閉じる |
| `<C-w>o` | 他を全て閉じる |
| `<C-w>=` | 均等サイズ |

### タブ操作

| キー | 動作 |
|------|------|
| `gt` / `gT` | 次/前のタブ |
| `:tabnew` | 新規タブ |
| `:tabclose` | タブを閉じる |

### キー表記

| 表記 | 意味 |
|------|------|
| `<C-x>` | Ctrl + x |
| `<CR>` | Enter |
| `<leader>` | スペース |
